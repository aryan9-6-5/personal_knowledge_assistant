# Personal Knowledge Assistant
A RAG system for querying research papers and documents.

## Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Set up `.env` with API keys.
4. Run: `streamlit run app.py`

## Usage
- Upload PDFs/text files or enter URLs.
- Ask questions in the chat interface.
- View answers with source attribution.

## Tech Stack
- Streamlit, LangChain, Groq, Hugging Face, Pinecone
